Function Documentation¶

HAPError HAPPlatformKeyValueStoreSet(HAPPlatformKeyValueStoreRef keyValueStore, HAPPlatformKeyValueStoreDomain domain, HAPPlatformKeyValueStoreKey key, const void *bytes, size_t numBytes)¶

Sets the value of a key in a domain to the contents of a buffer.

Return

kHAPError_None If successful.

Return

kHAPError_Unknown If persistent store access failed.

Parameters

  • keyValueStore: Key-value store.

  • domain: Domain to modify.

  • key: Key to modify.

  • bytes: Buffer that contains the value to set.

  • numBytes: Length of bytes buffer.
