Detailed Description¶

For hardware-based authentication, byte blocks from a HomeKit controller are forwarded to an accessory’s Apple Authentication Coprocessor via its I2C interface, and vice versa. The platform developer does not need to understand the format or meaning of the data being forwarded. Expected behavior: Apart from a few functions for enabling/disabling the Apple Authentication Coprocessor, two functions must be implemented: one for writing and one for reading a byte block over I2C. Apple Authentication Coprocessors 2.0C and 3.0 are supported by the HAP Library.
