Definition (PAL/HAPPlatformCamera.h)¶

  • Program Listing for File HAPPlatformCamera.h

Detailed Description¶

Cameras (with or without camera event recording) and video doorbells need to implement the HAPPlatformCamera.h API. See IP Camera for more information.

Expected behavior:

Default implementation for Raspberry Pi that uses the ADK’s own implementation of SRTP etc. (HAP-RTP). It requires implementation, and possibly adaptation, of the following helper APIs:

  • HAPPlatformCameraInput+Init.h / HAPPlatformCameraInput.h

  • HAPPlatformMicrophone+Init.h / HAPPlatformMicrophone.h

  • HAPPlatformSpeaker+Init.h / HAPPlatformSpeaker.h
