Function Documentation¶

HAP_RESULT_USE_CHECK HAPError HAPPlatformCameraInputTakeSnapshot(HAPPlatformCameraInputRef cameraInput, HAPPlatformCameraSnapshotRef snapshot, uint16_t width, uint16_t height, HAPPlatformCameraSnapshotReader *_Nonnull *_Nonnull snapshotReader)

Takes a camera snapshot.

Return

kHAPError_None If successful.

Return

kHAPError_OutOfResources If out of resources.

Return

kHAPError_Unknown If an I/O error occurred.

Parameters

  • cameraInput: Camera input.

  • snapshot: Snapshot state.

  • width: Width of the requested image.

  • height: Height of the requested image.

  • [out] snapshotReader: Reader for the image.
