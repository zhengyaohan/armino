Function HAPPlatformWiFiManagerApplyConfiguration¶

  • Defined in File HAPPlatformWiFiManager.h

Function Documentation¶

HAPError HAPPlatformWiFiManagerApplyConfiguration(HAPPlatformWiFiManagerRef wiFiManager, const char *ssid, const char *passphrase, const char *regulatoryDomain, HAPPlatformWiFiManagerCookie cookie, uint32_t updateStatus, bool restartWiFi)¶

Applies the WiFi configuration. Persists the WiFi credentials passed in. #if HAP_FEATURE_ENABLED(HAP_FEATURE_WIFI_RECONFIGURATION) Restarts the WiFi interface if restartWiFi is set to true #endif

Return

kHAPError_None Applying the configuration was successful.

Return

kHAPError_Unknown Applying the configuration fails.

Return

kHAPError_OutOfResources Restarting wifi fails as the buffer passed to store results is small.

Parameters

  • wiFiManager: WiFi manager.

  • ssid: The SSID of the WPA/WPA2 personal WiFi network. If NULL, do not write the network info.

  • passphrase: If the WiFi network is secured, the network’s passphrase credential: 8-63 printable ASCII characters or 64 hexadecimal digits.

  • regulatoryDomain: Optional regulatory domain in which the device is operating: NULL-terminated country code (ISO 3166-1 alpha-2). #if HAP_FEATURE_ENABLED(HAP_FEATURE_WIFI_RECONFIGURATION)

  • cookie: Cookie value sent from the controller in WiFi reconfiguration request

  • updateStatus: UpdateStatus processed while handling fail safe WiFi reconfiguration request

  • restartWiFi: Flag set to indicate whether WiFi should be restarted #endif
