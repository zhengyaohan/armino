Detailed Description¶

An IP stack that supports several TCP connections is the basis for all HAP over IP accessories. The HAP server uses a stream abstraction for its IP connections. The PAL implementation must wrap the IP stack functionality for open, close, read, and write. Expected behavior: An IP protocol stack must meet the following requirements:

  • A POSIX dual IPv4 / IPv6 stack based on Berkeley sockets, also known as BSD sockets

  • Concurrent IPv4 and IPv6 connections must be supported

  • At least 8 concurrent open connections must be supported
