Detailed Description¶

Many HomeKit IP accessories are based on Wi-Fi and need some specific support for taking part in Wi-Fi networks. Expected behavior: Following functions need to be implemented, to enable a Wi-Fi accessory to join and to leave a Wi-Fi network.
